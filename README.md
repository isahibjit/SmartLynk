## ğŸ”§ Installation & Setup

1. **Clone the repo**

   ```bash
   git clone https://github.com/isahibjit/SmartLynk.git
   ```

2. **Navigate into the project**

   ```bash
   cd SmartLynk
   ```

3. **Install dependencies**

   ```bash

   ```
